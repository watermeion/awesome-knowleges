# Posix Command

## 特殊符号

　　$#：传入脚本的参数个数；

　　$0:  脚本自身的名称；　　

　　$1:  传入脚本的第一个参数；

　　$2:  传入脚本的第二个参数；

　　$@: 传入脚本的所有参数；

　　$*：传入脚本的所有参数；

　　$$:  脚本执行的进程id；
  
　　$?:  上一条命令执行后的状态，结果为0表示执行正常，结果为1表示执行异常；
